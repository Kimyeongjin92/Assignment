### 4. 양의 정수를 매개변수로 1에서부터 매개변수 값까지 홀수를 더해서 그 결과를 리턴하는 함수 oddSum을 작성 oddSum(100) for필수.
  oddSum <- function(limit){
    sum<-0
    for (i in 1:limit){
      if (i %% 2 == 1)
        sum <- sum +i
    }
    return(sum)
  }
oddSum(100)
